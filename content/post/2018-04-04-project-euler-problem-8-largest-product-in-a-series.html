Find the thirteen adjacent digits in the 1000-digit number that have the greatest product. What is the value of this product?

Gọi N là số một tự nhiên cho truớc (N > 1).

Gọi k là số chữ số (k <= nchar(N)) dùng để tạo các cặp số liền kề từ N.

Ví dụ: N = 15342, k = 3, ta có thể tạo các cặp số sau đây: [1, 5, 3], [5, 3, 4], [3, 4, 2].

Hàm make_series() sau đây được sử dụng để tạo các cặp số từ N và k.

make_series <- function(N, k) {
  stopifnot(k <= nchar(N))
  N <- strsplit(as.character(N), split = "")[[1]]
  N <- as.integer(N)
  first <- 1
  last <- length(N) - k + 1
  lapply(first:last, function(x) N[x:(x + k - 1)])
}


make_series(15342, k = 3)
